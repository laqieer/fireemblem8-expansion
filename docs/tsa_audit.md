# Fire Emblem: The Sacred Stones — TSA (tilemap) data audit

A **TSA** is a GBA BG tilemap: `u16` entries `tile(10b) | hflip | vflip | pal(4b)`.
Stored either headered (`.tsa.bin`, 2-byte `[w-1,h-1]` prefix) or headerless
(`.map.bin`); both are committed uncompressed and LZ-compressed at build time.
Each TSA is rendered for review by `scripts/gfxtools/tsa_preview.py` into
`preview/tsa/**.png` (preview-only, never built into the ROM).

This document audits every TSA group: its **role** (traced from the decompiled
code that loads it), its **size**, its **tileset** and **palette**, a short
**content** description, and whether the name matches. Generated by an audit
workflow over 126 groups / 430 TSA assets.
